HOW TO DEPLOY:

  OPTION 1: AUTOMATIC (RECOMMENDED)
  ==================================
  1. git add . && git commit -m "Deploy to Cloudflare" && git push
  2. Go to https://dash.cloudflare.com/
  3. Pages → Create Project → Connect to Git
  4. Select repository
  5. Build Command: npm run build
  6. Build Output: dist
  7. Click Save and Deploy
  8. Wait 1-2 minutes
  9. Your app is live! 🎉

  OPTION 2: MANUAL
  ===============

  5. Your app is live! 🎉
