🚀 DATA & STORAGE

Where is data stored?
All data is stored in your BROWSER's localStorage
- No backend needed
- No database needed
- Works for small teams (< 10 users)
- Each device/browser has separate data

How to backup data?
Open browser console (F12) and run:
localStorage.getItem('projects')
localStorage.getItem('tasks')

How to clear data?
localStorage.clear()

❓ FAQ

Q: What if I get a blank page?
A: Check browser console (F12) for errors. Try clearing cache.

Q: Can I use a custom domain?
A: Yes, in Cloudflare Pages settings → Custom domains.

Q: How do I update after deployment?
A: Just push to main branch, Cloudflare auto-deploys.

Q: Can multiple users use it at the same time?
A: Yes, but each device stores its own data (no sync).

Q: What's the cost?
A: FREE for up to 500 deployments/month and unlimited requests.

Q: Can I rollback to a previous version?
A: Yes, in Cloudflare Pages → All deployments → Rollback.

🆘 TROUBLESHOOTING

Build won't start:
  → Check "Build command" is: npm run build
  → Check "Build output dir" is: dist
  → Clear Cloudflare cache and retry

Blank page after deploy:
  → Check _redirects file exists in root
  → Verify index.html exists in dist/
  → Check browser console for JS errors

Data not showing:
  → Refresh the page (F5)
  → Clear browser cache (Ctrl+Shift+Delete)
  → Try opening in incognito mode

Still stuck?
  → Check build logs in Cloudflare Dashboard
  → Open browser console (F12) for error details
  → Review README.md for more information
